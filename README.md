# README

## **Description**

Api for creating warehouses and reserving time slots in them.

## **Prerequisites**

- Ruby 3.1.2 installed on the system

## **How To Start**

1. `bundle install`
2. `rails db:create && rails db:migrate && rails db:seed`
3. `rails s`

## Enhancements

- Adding timezone to the warehouse
- Adding monitoring service
